# 名字的由來
鴻漸是紅(石)建(築)的諧音
我希望能讓玩家在伺服器中發揮自己的想象力
建立自己和朋友的家園 甚至是城市
和朋友一起討論紅石機器 把想法付諸行動

1. 分流系統
  伺服器有三大分流 - 家園 冒險 紅石
  分流之間物資, 玩家經驗等互相流通, 玩家可以隨時切換分流

  1. 家園分流
     - 地形: 真實地形, 讓你的家園看上去更加自然
     - 不開放終界和地獄    
     - 不能使用TNT
     - 簡單模式
     - 永不洗白
     - 地圖邊界: 100k x 100k
     - 理想用途: 村民交易中心 玩家建築群 各種交流建築 活動中心

  2. 冒險分流
     - 主要用途: 探索終界和地獄
     - 困難模式
     - 地圖邊界: 200k x 200k
     - 每半年洗白一次
     - 理想用途: 生怪塔 探索終界城

  3. 紅石分流
     - 紅石大佬們發揮實力的地方
     - 小至刷石機 大至區塊吞噬者 蓋在這裏就對了
     - 紅石玩家可以在這裏建造各種生產物品的機器
     - 為建築師提供原材料 創建家園
     - 簡單模式
     - 永不洗白
     - 地圖邊界: 500k x 500k

2. 白名單系統
  - 玩家需要在申請並通過驗證後才能進入伺服器游玩
  - 預防惡意玩家刻意搗亂

3. 伺服器備份
  - 在每分每秒, 伺服器的每一隻生物被殺 方塊更改 拾取掉落物都會有記錄, 以此監管玩家行爲
  - 伺服器每半天就會將地圖檔備份一次 在地圖檔有bug時也能馬上更換, 以保護玩家的建築

# 伺服器功能
- 選單 (例如: GUI市場)
- 連鎖挖礦 鋤頭快速收割
- 掉落物堆叠 自動拾取
- 物品整理 額外的6x9背包
- 領地 傳送
- 在聊天頻道顯示手上物品
- 游玩記錄排行榜 (例如: 方塊挖掘數)
- 公會 每日任務
- 防透視

# 伺服器資訊
開服時間預定在1.17版本推出後一段時間 需要時間更新與測試插件版本
- 伺服器版本 : 1.17
- IP：hjmc.net
- CPU：i10850k
- 最高遊玩人數：不限
- 開放時間：24小時

# 伺服器規則
1. 不破壞其他玩家游戲體驗
2. 不惡言相向
3. 待新增
